# ⚗️Mercury

Mercury is a collection of UDFs for Hive.

| Category   | Description                                 |
|------------|---------------------------------------------|
| collect    | Deals with collections (Array, Map, Struct) |
| statistics | Deals with statistical operations           |

```
.

│ └── ...  
├── statistcs  
│ ├── JaccardSimilarity  
│ └── ...   
└── ...  
```
